# geospatial Package
> This package has the main functions needed to get, transform and produce a mastertable of open street data based per city and country on a pre-defined meshgrid of points.
## Overview
This package uses data from open street map and it allows you to calculate features based on different amenities like point of interests (restaurants, atm, etc) subways or highways.
## Usage
The package is divided into 4 main group of functions:

- cleaning: has functions to clean the data
- feature: has functions to generate new features based on geospatial data
- functions: this is a utility group of functions
- validation: contains functions for data validation

```sh
$  pip  install  segai.geospatialde
```
