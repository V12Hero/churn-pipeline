# worldpop Package

> This is a compendium of functions used for the extractions and transformation of the population density per point in the meshgrid, the data is 1km squared at it's minimum resolution.

## Overview

The main functions used are:

- transform_urbanicity_primary
- extract_world_pop
- get_h3_index
- get_h3_neighbours
- geopandas2spark
- pandas2spark
- create_features_urbanicity

## Usage

```sh
$  pip  install  segai.worldpopde
```
